# aischedule-xaufe 西安财经大学小爱课程表导入工具

## 使用

前往 小爱课程表-课程表设置-导入导出-教务网站导入 选择学校 “西安财经大学”，适配项目 “教务管理系统”，进入页面后按操作提示即可。
